# Werkplek 2.0

Hier is de log en links naar scripts van mijn WP2.0 installatie.

**Notes**

Je bent als ontwikkelaar local admin. Dat is _niet_ je eigen account, maar je krijgt credentials om als local admin mee in te loggen (`./administrator`/`Welkom08`). Nadat je bent ingelogd als gebruiker - met je Azure account - kun je je Azure account toevoegen aan de local admins group:

```
net localgroup administrators AzureAD\AndelMarcvan /add
```

Bron: [link](https://superuser.com/questions/982336/how-do-i-add-azure-active-directory-user-to-local-administrators-group)

**Choco**

Eerst heb ik [chocolatey](https://chocolatey.org/) geïnstalleerd. En vervolgens daarmee de rest:

`choco feature enable -n useFipsCompliantChecksums`

choco install dropbox -y
choco install keepass.install -y
choco install git -y
choco install putty.install -y
choco install vscode -y
choco install slack -y
